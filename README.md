Este projeto foi desenvolvido como parte da atividade da disciplina de Interação Humano Computador e UX.

## Objetivo

Compreender a criação de aplicações utilizando Blazor, com foco em componentes e reutilização.

## Funcionalidades

- Aplicação Blazor funcional
- Componente Counter reutilizado na Home
- Uso de parâmetros com [Parameter]
- Incremento personalizado do contador

## Funcionamento

- Na página Home, o contador incrementa de 100 em 100
- Na página /counter, o contador incrementa de 3 em 3

## Tecnologias

- .NET
- Blazor
- C#
- Razor
